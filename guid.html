GM Tool 가이드 문서

회원 검색

  • 회원 검색은 ID 와 닉네임 그리고 전화번호로 검색이 가능 합니다.
  • ID 검색 인 경우 해당 아이디가 존재 한다면 바로 사용자 정보 화면으로 이동 합니다.
  • 닉네임과 전화번호 검색인 경우 중복 값이 존재 할 수 있으므로 ID List 가 조회됩니다.

 

로그인 제한

  • 사용자 정보 > 사용자 검색 > 현재 정보 > 로그인 제한 버튼
  • 부정한 행위를 한 사용자의 경우 로그인을 제한 할 수 있습니다.
  • 제한등급 제한일수 사유를 입력 하신 후 제한 버튼을 클릭 하시면 해당 계정은 로봇킹 로그인이 제한 됩니다.

 

탈퇴

  • 사용자가 탈퇴를 원하는 경우 GM Tool 에서 탈퇴를 시킬 수 있습니다.

 

현재정보, 보유리스트, 소셜정보

  • 사용자의 현재정보, 보유리스트(무기, 파츠, 파일럿 등등), 소셜정보(우편함, 친구리스트) 등의 정보를 알 수 있습니다.

 

정보 변경

  • 사용자 정보 > 사용자 검색 > 현재 정보 > 변경 버튼
  • 사용자의 정보를 변경 할 수 있습니다.
  • 변경 사유는 필수 값이므로 제일 먼저 작성해 주시길 바랍니다.
  • 사용자의 정보를 변경 할 수 있습니다.

 

결제 정보

  • 사용자의 USN_ID 또는 주문번호를 통해서 결제 정보를 확인 할 수 있습니다.

 

메세지 발송

  • 특정 사용자나 전체 사용자에게 아이템 메세지를 발송 할 수 있습니다.
  • 기본 만료일은 3일 입니다.
  • 전체 발송일 경우 전체발송 체크박스를 선택해 주세요

 

이벤트 관리

  • 로봇킹의 이벤트를 관리 할 수 있습니다.
  • 최초 화면은 현재 적용 중인 이벤트 목록을 표시 합니다.
  • 이벤트가 중복 될 경우 마지막으로 등록한 이벤트가 적용 됩니다.

 

업무처리

  • 특정 GM ID 의 GM Tool 변경이력을 조회 할 수 있습니다.
